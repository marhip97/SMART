# Variabel: handelspartnervekst

| Felt | Verdi |
|---|---|
| **ID** | handelspartnervekst |
| **Navn** | Handelspartnernes BNP-vekst |
| **Type** | Betingingsvariabel |
| **Enhet** | Prosent, år/år-vekst |
| **Kilde** | Norges Bank Data, serie TPGDP |
| **Frekvens** | Kvartalsvis |
| **Tilgjengelig fra** | Ca. 1995K1 |
| **Publiseringsforsinkelse** | Ca. 60–90 dager etter kvartalsslutt |

## Definisjon

Vektet gjennomsnitt av BNP-vekst hos Norges viktigste handelspartnere, publisert av Norges Bank som del av pengepolitikkrapporten.

## Modellrelevans

Norsk eksport (utenom olje) er sterkt avhengig av internasjonal etterspørsel. Handelspartnervekst er en ledende indikator for norsk BNP Fastlands-Norge. Inngår som eksogen variabel i AR-X og VAR.
