# Variabel: lonnsvekst

| Felt | Verdi |
|---|---|
| **ID** | lonnsvekst |
| **Navn** | Lønnsvekst (årslønn, alle næringer) |
| **Type** | Målvariabel |
| **Enhet** | Prosent, år/år-vekst |
| **Kilde** | SSB Statistikkbanken, tabell 11417 |
| **Frekvens** | Årlig |
| **Tilgjengelig fra** | 1997 |
| **Publiseringsforsinkelse** | Ca. 6–8 måneder etter årslutt (endelig tall) |

## Definisjon

Gjennomsnittlig årslønnsvekst for alle næringer samlet, målt som prosentvis endring fra foregående år. Basert på SSBs lønnsstatistikk.

## Kjente brudd

- Årsstatistikken er basert på a-meldingen fra 2015; før dette på andre administrative registre. Mulig metodisk brudd rundt 2015.
- Pandemien (2020) ga statistisk støy pga. endret næringssammensetning.

## Modellrelevans

Kobler arbeidsmarked og inflasjon (Phillips-kurve). Lav frekvens (årlig) begrenser modellkvaliteten på kortere horisonter. Inngår som forklaringsvariabel for KPI-JAE og som målvariabel. Modeller bør ta hensyn til at serien bare oppdateres én gang per år.
