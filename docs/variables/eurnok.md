# Variabel: eurnok

| Felt | Verdi |
|---|---|
| **ID** | eurnok |
| **Navn** | EUR/NOK valutakurs |
| **Type** | Betingingsvariabel |
| **Enhet** | NOK per EUR (nivå) |
| **Kilde** | Norges Bank Data, serie EURNOK |
| **Frekvens** | Daglig (aggregeres til månedlig gjennomsnitt) |
| **Tilgjengelig fra** | 1999-01 (EUR-innføring) |
| **Publiseringsforsinkelse** | Ingen (markedspris) |

## Definisjon

Midtkurs EUR/NOK publisert av Norges Bank. En høyere verdi betyr svakere krone.

## Modellrelevans

Svak krone øker importprisene og dermed KPI. Sterk sammenheng med oljepris (negativ: høy oljepris gir sterkere krone). Inngår som forklaringsvariabel for KPI, KPI-JAE og BNP Fastlands-Norge.
